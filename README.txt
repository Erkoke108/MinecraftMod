# MinecraftMod

Este es un mod para Minecraft desarrollado usando Forge y Java. El proyecto está diseñado para ser compatible con Minecraft 1.20.1.

## Características

- Mod básico de ejemplo con estructura inicial configurada.
- Estructura lista para agregar contenido como bloques, ítems, entidades, etc.

## Requisitos

- [Java JDK 17 o superior (recomendado JDK 19)](https://adoptium.net)
- [Gradle 8.13](https://gradle.org/)
- [Minecraft Forge MDK 1.20.1](https://files.minecraftforge.net/)
- Visual Studio Code con la extensión de Java (Extension Pack for Java)

## Uso

### Compilar el proyecto

Desde la terminal integrada de Visual Studio Code:

```bash
./gradlew build
